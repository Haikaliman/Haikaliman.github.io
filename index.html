Expertise

I focus on making games at Roblox.

I'm using the Roblox platform to publish my game and use Roblox studio to make the game. I ben expose to Roblox since 2016 and since that day it has got me interested to make a game and learn to code.

  • Game planning

    When I making a game I usually do the planning first to determine what kind of game I would like to make. How it should work, what kind of story it had, and also how it can make the user gain the same experience as I do.

  • Game Map planning

    I usually used paint as my ma planning. At paint, I will draw the layout of the map, make sure to mark any spot in which I want to put prop/deco item and design it to make sure that it is fixed.

  • Modeling

    To build any model in Roblox you have so many options. One of the software to make your model is by using the robot studio itself. It has a ton o plugins to help you do the modeling. One of my models in the Roblox game is a simple island that I make from scratch.

  • Lua programming

    Lua programming is a high-level programing language. Lua is an important component when you want to make the game in Roblox. It will involve you to script any part or component of your model to make it work such as a tool.

  • Game marketing

    Making a Roblox game is one of the ways to gain money. The game will make a lot of money if the developers know what to do with their games. This is where game marketing comes in. To make money in the Roblox game you must create game passes for the user to buy and do advertisements. To check your revenue in Roblox, you can just check it at developer stats.
